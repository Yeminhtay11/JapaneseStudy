### Note on the JLPT N4 Level

##   ～ほど　文法
It　shows the degree of action (動作＝どうさ）or state　（状態　＝　じょうたい）。（~ လောက် in Burmese) 

pattern
Ｖ（　普通形　／　ナイ　形）＋ほど
いAdj（普通形）　＋ほど
Ｎ＋　ほど


Remarks
　カジュアルな　言い方は『～　くらい』
後ろに名詞　がくる場合「～ほどのN]　となる

動詞　文
１．今　家の近で　工事をしていて、　集中できないほど　うるさいです。

い形　
２．彼の気持ちが痛いほど　わかる。

名詞文
３．　今日は昨日ほど寒くない。
